Easy Calculator

Intro

Easy calculator is a tool written in C# which aims for a more efficient way to perform calculations, compared to a traditional calculator. Its goal is to provide a way for the user to make calculations with as little actions as possible. This is useful for people who have to make a lot of calculations manually.

The main features which sets this tool apart from a traditional calculator are:
  • You can perform a lot of manual operations by simply entering a number and pressing the Enter on your keyboard.
  • Ability for the user to remember the number input, simplifying the task even further; Now you can repeatedly press the Enter key to perform the same calculation multiple times.

User Interface

This is what the user interface looks like:

As you can see, the operator choice (plus, minus, multiply, divide by) is always remembered. Its advantage being the user not having to enter the operator symbol themselves. And the user has the ability to remember his input so that he only requires to press the Enter button to repeat the previous calculation.

The clear button clears the user input and the output of the tool, so that the user may start from scratch again.

The undo button removes the last operation performed on the formula.

The formula window displays the complete formula of the calculation made by the user.

On the very buttom is the output window for the answer. In case the application runs into a problem, it will display an error here instead of an answer.

Technical details

Easy calculator is a .NET Winforms project. Winforms is a framework designed to provide an easy way for developers to build user interfaces for Windows applications. In a similar way to Javascript, Winforms likes to add event handlers to components such as Buttons and Text fields.
